**2024 Presidential Candidate Fundraising Dashboard**

I created a dashboard in Power BI based on the 2024 Presidential Candidate Fundraising Data showing contributions received by parties and candidates by month, as well as sources of those contributions by location. I cleaned and manipulated the data in Power Query to make them usable for visualizations.  

Data: Campaign Finance Data from the Federal Election Commission (https://www.fec.gov/data/browse-data/?tab=raising).

Usage: Download the .pbix file and open in Power BI to view and interact with dashboard.
